## 安装与启动

### 1. 安装应用

1. 双击 `BiliManga_0.1.0_x64-setup.exe`
2. 选择安装位置（默认：`C:\Program Files\BiliManga`）
3. 点击"安装"
4. 等待安装完成

### 2. 首次启动

安装完成后：
- 从开始菜单找到"BiliManga"
- 或双击桌面快捷方式

首次启动会看到欢迎界面。

## 获取 Cookie

### 为什么需要 Cookie？

Cookie 是你的登录凭证，应用需要它来访问你的已购漫画列表。

### 获取步骤（Chrome 浏览器）

#### 方法 1：手动复制（推荐）

1. **打开 Chrome 浏览器**
   - 访问 https://manga.bilibili.com

2. **打开开发者工具**
   - 按 `F12` 键
   - 或右键 → 检查

3. **切换到 Network 标签**
   - 点击顶部的"Network"标签
   - 勾选"Preserve log"

4. **登录（如果未登录）**
   - 点击右上角登录按钮
   - 使用手机扫码或账号密码登录

5. **找到请求**
   - 在 Network 列表中找到任意请求
   - 推荐找 `manga.bilibili.com` 开头的

6. **复制 Cookie**
   - 点击请求
   - 切换到"Headers"标签
   - 找到"Request Headers"部分
   - 找到"Cookie:"行
   - 右键 → Copy value

7. **Cookie 示例格式**
   ```
   SESSDATA=xxx; bili_jct=xxx; DedeUserID=xxx; ...
   ```
   应该是很长的一串字符

#### 方法 2：使用脚本（高级）

如果你熟悉命令行，可以使用项目提供的测试脚本。

## 查看已购漫画

### 1. 保存 Cookie

启动应用后：

1. **找到 API 测试面板**
   - 滚动到页面底部
   - 看到"🧪 API 测试面板"

2. **粘贴 Cookie**
   - 在"Cookie 字符串"文本框中粘贴
   - 点击"🔒 保存到密钥环"按钮

3. **等待成功提示**
   - 看到"✓ Cookie 已安全保存到系统密钥环"
   - Cookie 已保存，下次启动会自动加载

### 2. 查看已购列表

1. **点击导航栏**
   - 点击左侧导航的"📚 我的已购"

2. **等待加载**
   - 首次加载可能需要几秒钟
   - 看到漫画封面和标题

3. **浏览漫画**
   - 网格布局显示所有已购漫画
   - 每张卡片显示：
     - 封面图片
     - 漫画标题
     - 已购章节数
     - 最新话数
     - 自动购买状态（如果开启）

4. **加载更多**
   - 滚动到底部
   - 点击"加载更多"按钮
   - 查看更多漫画

5. **刷新列表**
   - 点击右上角"🔄 刷新"按钮
   - 重新加载最新数据

## 常见问题

### Q1: 安装时显示"Windows 已保护你的电脑"怎么办？

**A**: 这是正常的 Windows SmartScreen 警告，因为应用未经过微软签名。

**解决方法**：
1. 点击"更多信息"
2. 点击"仍要运行"
3. 应用会正常安装

### Q2: Cookie 保存失败怎么办？

**A**: 可能的原因和解决方法：

1. **Cookie 格式不正确**
   - 确保从浏览器完整复制
   - 确保包含 `SESSDATA` 字段

2. **Windows 凭据管理器被禁用**
   - 打开控制面板 → 凭据管理器
   - 确保可以正常访问

3. **权限不足**
   - 右键应用 → 以管理员身份运行

### Q3: 已购列表为空怎么办？

**A**: 检查以下几点：

1. **Cookie 是否有效**
   - 在 API 测试面板点击"1️⃣ 检查登录状态"
   - 应该显示"登录成功"

2. **是否真的有已购漫画**
   - 访问官方网站确认
   - https://manga.bilibili.com/account-center

3. **尝试刷新**
   - 点击"🔄 刷新"按钮

### Q4: Cookie 多久会过期？

**A**: 
- 通常有效期约 30 天
- 过期后会提示"会话过期"
- 需要重新获取并保存

### Q5: 封面图片无法显示怎么办？

**A**: 
- 部分图片可能加载较慢
- 刷新页面重试
- 检查网络连接

### Q6: 如何删除保存的 Cookie？

**A**: 
1. 在 API 测试面板
2. 点击"🗑️ 删除保存"按钮
3. Cookie 会从系统密钥环中删除

### Q7: 数据存储在哪里？

**A**: 
- **Cookie**：Windows 凭据管理器（系统加密）
- **数据库**：`%APPDATA%\BiliManga\` （SQLCipher 加密）
- **缓存**：`%APPDATA%\BiliManga\cache\` （临时文件）

### Q8: 如何完全卸载？

**A**: 
1. **卸载应用**
   - 设置 → 应用 → BiliManga → 卸载

2. **清理数据（可选）**
   - 打开文件资源管理器
   - 输入 `%APPDATA%`
   - 删除 `BiliManga` 文件夹

3. **清理 Cookie（可选）**
   - 控制面板 → 凭据管理器
   - Windows 凭据 → 找到"BiliManga"
   - 删除

## 💡 使用技巧

### 1. 保持 Cookie 有效

- 定期访问官方网站（保持登录状态）
- Cookie 过期前会有提示

### 2. 管理缓存

- 应用会自动管理图片缓存
- 缓存有容量上限，自动清理过期文件
- 手动清理：删除 `%APPDATA%\BiliManga\cache\`

### 3. 快捷键（计划中）

目前版本暂不支持快捷键，未来版本会添加。

## 🔒 安全提示

1. **保护你的 Cookie**
   - Cookie 等同于登录密码
   - 不要分享给他人
   - 不要截图包含 Cookie 的画面

2. **定期更换密码**
   - 建议定期更换 B 站密码
   - 更换后需要重新获取 Cookie

3. **注意网络安全**
   - 不要在公共 WiFi 下使用
   - 使用受信任的网络
