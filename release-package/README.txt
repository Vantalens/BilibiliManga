# BiliManga v0.1.0 - Windows 安装包

欢迎使用 BiliManga！

---

## 快速开始

### 1. 安装

双击 `BiliManga_0.1.0_x64-setup.exe` 开始安装。

### 2. 获取 Cookie

打开 Chrome → 访问 https://manga.bilibili.com → F12 → Network → 复制 Cookie

### 3. 使用

启动应用 → 粘贴 Cookie → 点击"📚 我的已购"

## ⚠️ 重要提示

1. 首次运行可能提示"Windows 已保护你的电脑" → 点击"更多信息" → "仍要运行"
2. Cookie 有效期约 30 天，过期后需要重新获取
3. 应用仅供查看已购内容，不支持购买/解锁功能

---

## 系统要求

- Windows 10/11 (64-bit)
- 2 GB RAM
- 100 MB 磁盘空间
