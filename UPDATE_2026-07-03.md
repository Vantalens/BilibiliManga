### 1. 核心 API 实现

**`check_login_status()`** - 登录态检查
```rust
// 使用 B站主站 nav 接口
https://api.bilibili.com/x/web-interface/nav

// 错误码处理
- code=-101: 未登录
- code=0: 已登录，返回 uid
```

**`fetch_purchased_comics()`** - 已购漫画列表
```rust
// 端点
POST /twirp/user.v1.User/GetAutoBuyComics?device=pc&platform=web&nov=27

// 请求体
{"page_num": 1, "page_size": 15}

// 必需 Header
- Cookie: SESSDATA=...
- Referer: https://manga.bilibili.com/account-center
- x-bili-manga-from: c-int-v1

// 错误码处理
- code=0: 成功
- code=-6: 会话过期（来自 RSSHub 生产观察）
- code=99: 上下文不匹配
```

**错误码完整处理**
- `-101`: "not logged in - please visit official login page"
- `-6`: "session expired - please re-login via official page"  
- `99`: "request context invalid - may need browser session"

### 3. 关键发现记录

#### 鉴权机制
- **Web 端**：Cookie (`SESSDATA` 为核心)
- **必需 Header**：`x-bili-manga-from: c-int-v1`
- **必需 Referer**：`https://manga.bilibili.com/account-center`
- **必需 Query**：`device=pc&platform=web&nov=27`

#### 图片访问机制
- **三段式**：`GetImageIndex` → `ImageToken` → 带 token 的图片 URL
- **不是** DRM/EME，是应用层授权 + URL 签名
- 图片格式：webp，分辨率：raw/hd/sd

#### 分页模式
- **页码分页**：`page_num` 递增，不是 cursor
- **无 total 字段**：持续翻页直到返回空数组或数量 < page_size
- **默认页大小**：15
