# BiliManga SECURITY

版本：v0.1.0
作者：Codex
状态：草稿
最后更新：2026-07-02

## 安全边界

- 本项目只处理用户已授权、已购买或公开可访问内容。
- 支付、充值、购买、解锁、权益变更由官方网页处理。
- 本地数据库全库加密，密钥使用系统密钥环或等价安全存储。
- 日志不得包含 Cookie、Token、账号标识、订单信息或图片真实鉴权 URL。

## 缓存策略

- 图片缓存只用于短期性能优化。
- 缓存不可导出，不提供永久离线包。
- 必须支持容量上限、过期清理和一键清除。


## 当前实现状态

- Rust 后端已提供 `initialize_secure_storage`，会通过系统密钥环生成或读取数据库密钥，不向前端返回密钥。
- Rust 后端已提供 `clear_image_cache`，只清理应用缓存目录下的图片缓存，不接受前端传入任意路径。
- SQLCipher 实际数据库文件验证已完成；真实书库/进度写入必须使用该加密数据库路径。
